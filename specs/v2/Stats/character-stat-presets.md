# Character Stat Presets

**This file provides stat value presets for common character archetypes. Use these as starting points for characters in role play scenarios.**

**Stat Scale Reference:**
- **Tension (0-100):** Stress/anxiety levels
- **Restraint (0-100):** Self-control and inhibition
- **Connection (0-100):** Emotional bond to partner
- **Desire (0-100):** Sexual attraction/motivation
- **Dominance (0-100):** Control/agency in situations
- **Loyalty (0-100):** Commitment to relationship
- **SelfRespect (0-100):** Self-worth and boundaries

## Using These Presets

**Guidelines:**
1. **These are starting points** - adjust based on specific scenario needs
2. **Stats evolve during scenarios** - use stat affinities from themes to adjust
3. **Character development** - stats should change as characters progress through scenarios
4. **Consistency** - use consistent stat patterns for recurring characters

**Common Adjustments:**
- Increase Tension when character is in stressful situations
- Decrease Restraint as character loses inhibition
- Decrease Connection as emotional distance grows
- Increase Desire as arousal builds
- Decrease Loyalty as betrayal occurs
- Adjust SelfRespect based on character's confidence and self-worth
